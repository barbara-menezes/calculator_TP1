## Objetivo: Criação de uma calculadora em Ruby usando RSpec

Foi desenvolvida uma calculadora com as seguintes operações básicas:
- Somatório
- Subtração
- Multiplicação
- Divisão

Gerar relatorio em HTML:
rspec unit.rb --format html --out relatorio.html
